# AI-система для анализа научных публикаций

Система для поиска и синтеза знаний из корпуса научных статей с использованием продвинутых техник RAG.

## Основные характеристики
- **Источник данных:** Open Access API (OpenAlex).
- **Архитектура:** Sentence Window Retrieval (нетривиальный RAG), обеспечивающий глубокий контекст ответов.
- **LLM:** Локальный запуск модели Qwen 2.5 (0.5B) через Ollama (полная автономность).
- **Интерфейс:** Streamlit чат-система.

## Метрики качества (Evaluation)
Для оценки системы использовались локальные замеры на тестовом наборе (Ground Truth):
- **Semantic Recall:** 0.7454 (оценка полноты извлечения контекста).
- **BERTScore (F1):** 0.8631 (семантическое сходство ответов с эталоном).

## Как запустить
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите Ollama и скачайте модель: `ollama pull qwen2.5:0.5b`
3. Соберите базу: `python src/indexer.py`
4. Запустите чат: `streamlit run src/app.py`

# Изменения по сравнению с изначальным кодом

- Сбор данных переведен на открытые источники через OpenAlex (open-access), без закрытых библиотек и внутренних данных.
- Загрузка PDF стала безопаснее: проверка типа контента, лимит размера, удаление битых файлов.
- Метаданные расширены: `title`, `year`, `doi`, `work_id`, `pdf_url`, `source=openalex`.
- Индексация упрощена и стабилизирована: явные проверки наличия PDF и пустых коллекций.
- UI Streamlit очищен: стабильная обработка ошибок, корректный вывод источников и контекстных окон.
- Оценка качества стала более устойчивой: `bert-score` необязателен, добавлены CLI-параметры.
- Добавлен утилитарный `parser.py` для извлечения текста из PDF.
